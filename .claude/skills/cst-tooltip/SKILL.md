## 핵심 규칙 요약

- 직업/액티브/패시브 스킬 모두 동일한 색상 규칙 적용
- 색상 표준 코드표 확인 (수치, 이름, 효과 타입별 색상 구분)
- `GenerateTooltip()` 파라미터에 색상 코드 적용
- 스킬 변경 5종 세트 중 툴팁도 반드시 포함
- 툴팁 색상 변경 시 `cst-localization`과 함께 다국어 텍스트도 수정
